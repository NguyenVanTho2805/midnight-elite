### 2. Cloudinary Upload Utility

**`src/lib/cloudinary.ts`** (file mới)

- `cloudinaryConfigured` boolean — graceful degradation khi chưa cấu hình env vars
- `uploadToCloudinary(file, folder)` — direct browser → Cloudinary REST API, không cần npm package
- `uploadMany(files, folder)` — upload nhiều file song song

**`.env`** — thêm:

### 3. API Routes — Cập nhật + Mới

| Route | Method | Mô tả |
|---|---|---|
| `/api/community/threads` | GET | Thêm `limit` param, `bookmarkedByMe` vào DTO |
| `/api/community/threads` | POST | Nhận `imageUrls`, `fileUrl`, `fileName` |
| `/api/community/threads/[id]` | GET | Reply trả về `imageUrls`, `likeCount`, `likedByMe`, `isOwn` |
| `/api/community/threads/[id]/reply` | POST | Nhận `imageUrls` (max 2), trả về `likeCount`, `likedByMe` |
| `/api/community/threads/[id]/bookmark` | POST | Toggle bookmark — upsert/delete `ThreadBookmark` |
| `/api/community/replies/[id]/like` | POST | Toggle reply like — upsert/delete `ReplyLike` |
| `/api/community/replies/[id]` | DELETE | Xoá reply (owner hoặc admin) |

## Cấu hình cần làm sau

Để tính năng upload ảnh/file hoạt động, cần update `.env`:

```
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME="<cloud_name>"
NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESET="<unsigned_preset>"
```

1. Tạo tài khoản miễn phí tại cloudinary.com
2. Settings → Upload → Add upload preset → Mode: **Unsigned**
3. Copy Cloud Name và Preset Name vào `.env`

Khi chưa cấu hình: nút Ảnh/Tệp vẫn hiển thị nhưng có cảnh báo vàng, không block đăng bài text.
