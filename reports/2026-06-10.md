## Notion Flat Design System — Quy tắc canonical

| Token | Value | Dùng cho |
|---|---|---|
| Canvas | `#ffffff` | Card background, main surfaces |
| Surface | `#f6f5f4` | Inset sections, disabled states, sidebar items |
| Hairline | `#e5e3df` | Tất cả borders |
| Brand blue | `#0068FF` | Primary buttons, links, active states |
| Brand navy | `#0a1530` | Hero bands (`.notion-hero-band`) |
| Button radius | `8px` | Buttons (NOT pill), inputs |
| Card radius | `rounded-xl` | Tất cả card containers |

**Neumorphic colors BỊ BAN trong non-admin code:**

| Cũ | Mới |
|---|---|
| `background: "#F0F5FF"` + `boxShadow: "Xpx Xpx #C5D0EA..."` | `background: "#ffffff", border: "1px solid #e5e3df"` |
| `boxShadow: "inset Xpx Xpx #C5D0EA..."` | `background: "#f6f5f4", border: "1px solid #e5e3df"` |
| `linear-gradient(145deg, #0055D4, #0042AA)` button | `background: "#0068FF", borderRadius: "8px"` |
| `#C5D0EA` border/stroke | `#e5e3df` hoặc `#c8c4be` |
| `#E5ECF8` progress track | `#e5e3df` hoặc `#f6f5f4` |
