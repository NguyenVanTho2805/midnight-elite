## 6. Thiết kế lại Admin — Anti-AI Slop

### Bối cảnh
User chia sẻ bài Wikipedia về AI Slop → yêu cầu redesign thoát AI slop patterns.

### Button style: Flat Modern
- Tất cả colored gradient buttons → bỏ neumorphism shadow → flat
- `hover:brightness-110` thay `hover:-translate-y-0.5`
- `active:scale-[0.98]` tactile feedback
- Filter buttons inactive → border thay inset shadow

### Sidebar refactor
- `NAV_ITEMS icon: string` (emoji) → `Icon: IconComponent` (griddy-icons SVG)
- Icons: `ChartBar`, `BookOpen`, `ClipboardList`, `Badge`, `UsersGroup`, `Wallet`, `Edit`
- Bottom links: xóa `🌐`, `👨‍🎓` emoji

### Decorative elements removed
- `animate-pulse` dots trên health cards (không semantic)
- Decorative dot trong role badge dashboard header
- `animate-pulse` green dot trong vinh-danh "Đang hiển thị" → static border

### Emoji cleanup (toàn bộ admin)
| Replaced | With |
|---|---|
| `🔔 Nhắc nhở` | `Nhắc nhở` |
| `⭐ Cấp badge`, `🏆 Bảng xếp hạng`, `📥 Import` | Text thuần |
| `✏️` edit buttons | `Sửa` |
| `🗑` delete buttons | `Xóa` |
| `👤 Hồ sơ học sinh` | `Hồ sơ học sinh` |
| `💰 📚` stats icons | Xóa icon |
| `🔴 Live` | `● Live` |
| `📄 📁` upload zone | `↑` arrow |

### Em-dash cleanup
- `Học sinh cần can thiệp — GPA...` → `:`
- `Tất cả học sinh đều ổn — không...` → `,`
- `Không thể xoá — còn X...` → `:`
- `Đã cấp toàn bộ badge — ${total}` → `,`
- `— Bạn` inline label → `(Bạn)`
- `Slug (URL) — không đổi` → `(không đổi)`

## 7. Hoàn thiện tính năng còn thiếu

### SePay — Xóa hoàn toàn
- Xóa: `api/webhook/payment`, `api/payment/create`, `api/orders/`, `api/orders/[id]/`
- Xóa student pages: `gio-hang`, `thanh-toan`, `thanh-toan/thanh-cong`, `lich-su-mua-hang`
- Schema: xóa `PendingOrder` model, `Order` model, relations
- Chạy `prisma db push` — DB đồng bộ
- Xóa `CartProvider` khỏi root layout
- `PopupBuyRequired`: thay "Mua ngay" → "Liên hệ admin để đăng ký"

### Badge Persistence — Lưu DB
- Thêm `UserBadge` model: `userId`, `badgeId`, `grantedAt`, `grantedBy`
- `POST/GET/DELETE /api/admin/badges`
- Vinh-danh page: load từ DB khi mount, save khi trao, check `userId:badgeId`

### Video Auto-Complete — YouTube 80%
- `VideoPlayer` nhận `onAutoComplete?: () => void` prop
- Khi lesson chưa xong: load YouTube IFrame API, poll `getCurrentTime/getDuration` mỗi 2s
- Khi xem >= 80% → gọi `markComplete` (fire 1 lần duy nhất)
- Lesson đã xong hoặc non-YouTube: plain iframe như cũ

### Email Domain
- Xóa `SEPAY_WEBHOOK_TOKEN` khỏi `.env`
- `EMAIL_FROM` hiện = `onboarding@resend.dev`
- Hướng dẫn: verify domain tại resend.com/domains → đổi `EMAIL_FROM="no-reply@domain.com"`
