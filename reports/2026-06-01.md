## 6. Admin /doanh-thu/ — Rewrite Enrollment-based

### Triết lý mới
**Doanh thu = Enrollment × course.price** (phù hợp thu tiền thủ công)

### API analytics rewrite
```
totalRevenue     = sum(enrollment.course.price)
totalEnrollments = count(enrollments)
totalStudents    = users với ít nhất 1 enrollment
totalCourses     = courseRevMap.size (khoá thực sự có học viên)
byCourse         = grouped revenue per course (không slice)
allEnrollments   = tất cả enrollments cho bảng + CSV
```

### Fixes
- Xóa `useOrders`, `COURSES` import, `todayRevenue`, `byDay` dead fields
- `PUT /api/orders/[id]` (NEW) — manual confirm lưu DB thật
- Stats: chỉ money cards có "đ"
- Filter dropdown từ `byCourse` (đủ)
- CSV export toàn bộ
- `TB/khoá` chia `byCourse.length`
- `displayTotal`, `displayEnrollments`, `chartMax` dùng `useMemo`
- Xóa `user.id`, `course.id` dead selects
